Learning Outcomes and Introduction

In this lab assignment you will learn about:

  • Introduction to NetBeans environment.
  • Editing, compiling and running a Java program.
  • Creating your first Java program

The following is a step-by-step guide that will show you how to create a project and run a simple java program with NetBeans

There are many other ways to run java programs with NetBeans, but this method is simple and will be useful later in the course. Log on to the computer with your Camosun login ID.

Task 1: Create a project

  1. Go to start --> all programs --> launch NetBeans
  2. In NetBeans go to File --> New Project
  3. In the New Project window, select Java as a category and Java Application as a project, then click Next
  4. Uncheck Main Class checkbox
  5. For the Project Name, choose a name that is meaningful. For example “COMP132Lab1”. Make sure there is no space in the name
  6. For the project location, pick a location on the M drive and create a folder that will hold all your projects. For example “NetBeansProject”. Make sure there is no space in the name
  7. Click Finish. The project will appear under the project tab in NetBeans
  8. To add class files to your project:
    1. Select the project, go to File --> New File
    2. In the New File window, select Java as a category, and Java Class as a File Type, then click Next
    3. Choose a meaningful name. For example IntroToNetBeans. Make sure there is no space in the name
    4. Type the code below into the file, then save.
      Note: the name of the file is the same as the name of the class.

Task 2: Compile and run your project

  1. Go to the folder you created previously to host your project and you will notice a set of folders and files. One of the folder is called src. This folder contains .java file. This is the file you created in the previous step. It is known as the source code
  2. Go to the menu Run-->Compile File. Check in the output tab ( in the bottom section of NetBeans and confirm that you have a message saying BUILD SUCESSFULLY. If you have syntax error in your code, the output will indicate this.
  3. Check the project folder again, and notice a “build” folder that has been added. Open the “build” folder, then “classes” folder. This folder contain the .class file. These file is the java bytecode. This is the code that will run in the virtual machine when you run your java program
  4. The difference between .class and .java will be explain in the course of the semester
  5. To run your program within NetBeans, go to menu Run --> Run Project. Notice the outcome in the output output window. It should read: 
    					Hello World
					This is my first Java Program
