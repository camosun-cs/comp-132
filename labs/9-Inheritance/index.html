Learning Outcomes and Introduction

In this lab assignment you will learn about:

In this lab you will create your first Object. And then write testers to use your classes.

Task 1: Create a Class Time

    Write a time class that contains the following fields: hours, minutes and seconds. The class should also have the following methods:

    • Set : to set the time
    • Write: to write the time
    • Equal: to compare the time in term of equality (returning true or false)
    • Less than: to compare the time in term of precedence (returning true or false)
    • Provide few cases where a class is initialized, using a constructor
    • Provide few cases where these methods are called.
      • Note: I might want to leave some hints for students.

Task 2: Create a Class Bank Account

    Write a simple class representing a bank account: the bank account has 2 fields: balance, interest rate

    • Provide a constructor for this class
    • Write a method that set a bank account
    • Write a method that get the balance
    • Write a method that get the rate
    • Write a method that update the balance
