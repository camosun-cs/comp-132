Learning Outcomes and Introduction

In this lab you will create two small programs using what you've learned about:

  • if, else, and switch statements
  • Conditional operators (&&, ||, !)
  • Good coding style

You will also learn how to create random numbers using Math.random()

Task 1: Days in a Month

Prompt the user to choose a year and a month. Use a switch statement with a nested if/else to determine the number of days in the month and print the result.

Remember that you can use the "fall-through" behavior of the switch statement to avoid code duplication for similar cases. Also, cases do not have to be in any particular order, so put them in the order that saves the most typing!

Do not use any of the built-in date handling functionality in Java.

February has a different number of days depending on the year; 28 days in common years, and 29 in leap years. Leap years occur in each year that is a multiple of 4, except those that are also divisible by 100.

Task 2: Play The Lottery

Write a program to run a simple lottery. The program should randomly generate a lottery number between 0 and 99, prompt the user to enter a guess, and determine the user's winnings according to the following rules:

  • If the user input matches the lottery number exactly, the award is $10,000.
  • If the user enters the correct digits, but in the wrong order, the award is $3000.
  • If the lottery number has two digits and the user's guess contains just one of them, the award is $1000.
  • In all other cases, there is no award.
Remember to check that the user input is within the range you expect for this lottery.
Generating random numbers

The code to generate a random number for this task is as follows:

int lottery = (int)(Math.random()*100);

Math.random() generates a random positive double value between 0 and 1 (eg. 0.1, or 0.75639). The value may be exactly 0, but will never be exactly 1. When multiplied by 100, this transforms it to a range from 0 up to but not including 100 (although it could be 99.99999). Any decimal portion of the value will be dropped when cast to an int, leaving you with a random integer value between 0 and 99 (inclusive).

You can use Math.random(), along with a little bit of math and type casting, to generate any range of random values that you need. For example, you could get a random multiple of 3 between 15 and 150 with 3 * ((int)(Math.random() * 46) + 5).
