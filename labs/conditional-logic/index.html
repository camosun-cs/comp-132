Learning Outcomes and Introduction

In this lab you will create three small programs using what you've learned about:

  • if, else, and switch statements
  • Conditional operators (&&, ||, !)
  • Good coding style

Task 1: Double The Number

Write a program that prompts the user to enter a number within the range of 0 to 100. Your program should output twice the amount given.

Your approach should be something like:

  1. Prompt the user to enter a number between 0 to 100.
  2. Check that the user entered a number in the correct range (and display an error message if appropriate).
  3. Multiply the given number by two.
  4. Display the result.

Task 2: Play The Lottery

Write a program to run a simple lottery. The program should randomly generate a two-digit lottery number, prompt the user to enter a guess, and determine the user's winnings according to the following rules:

  • If the user input matches the lottery numbers in exact order, the award is $10,000.
  • If the user enters the correct digits, but in the wrong order, the award is $3000.
  • If one digit in the user input matches a digit in the lottery, the award is $1000.
Remember to check that the user input is exactly two digits!

The code to generate a random number is as follows:

int lottery = (int)(Math.random()*100);

Task 3: Find My Team

Prompt the user to enter a team number. Use a switch statement to find the appropriate league name for that team and print it to the screen. The teams are grouped into the following leagues:

  • "Junior" league: teams 1, 2, 3, and 6.
  • "Amateur" league: teams 8, and 9.
  • "Special" league: teams 4, 7, and 11.
  • "Major" league: teams 5, 10, and 12 through 14.
  • Any other team number: "Unassigned".
