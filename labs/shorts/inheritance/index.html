  1. Task
  2. Submission

Task

Create a hierarchy of classes that extend your Person class from the previous lab.

Your submission should include the package containing the three new classes:

  • Student (subclass of Person)
  • GraduateStudent (subclass of Student)
  • Employee (subclass of Person)

Import your existing Person class rather than recreating it in this new package.

Your submission should demonstrate an understanding of:

  • how to use extends
  • using super() to invoke the parent's constructor
  • using inherited variables and methods
  • overriding inherited methods

Submission

Submit a ZIP archive of your package folder to the "Inheritance Exercise" assignment on D2L.

You do not need to include your Person class.

This assignment must be handed in individually, but you are encouraged to work together in groups and share code with one another to help you to understand the concepts!

You will receive a 100% completion mark as long as you hand in some code that demonstrates your understanding of the exercise.
