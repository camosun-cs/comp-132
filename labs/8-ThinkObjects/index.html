Learning Outcomes and Introduction

In this lab assignment you will learn about:

  • Use of Graphics Class
  • Use of String Class
  • Design Classes from other classes

In this lab you will work on thinking in Objects. Research, locate and use classes from Java library inorder to build new objects.

Task 1: Create a Class Flag

Write a Flag class that contains the following fields: X, y coordinates, width, height, and the color of the rectangle. The class should also have the following methods:

  • Constructor: to create a flag
  • Setters and Getters for all the attributes of the flag
  • Be able to change color of the rectangles in the flag
  • Be able to change the position of the rectangles in the flag
  • Be able to change the width of the rectangles in the flag
  • Be able to change the height of the rectangles in the flag
  • Create a method given the parameters needed to display a flag.

Task 2: Use the Class Flag

Test your class by instantiating Flag class and pass the proper information to draw

  • Draw the flag from Belgium

  • Draw the flag from Swiss
