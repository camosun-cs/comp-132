Learning Outcomes and Introduction

In this lab assignment you will practice:

  • Creating and using methods
  • Creating and modifying arrays
  • Passing parameters and receiving return values
  • Using multi-dimensional arrays

Task 1: Restructure "Play The Lottery"

  1. Recreate your "Play The Lottery" program from Lab 3 Task 2 using methods.

    Your program should be composed of at least 4 different methods:

    • a method that generates a lottery number
    • a method that prompts the user for a guess and ensures that their input is valid
    • a method that compares a guess to a lottery number and determines whether or not the guess was correct (or partially correct)
    • the main method which controls the flow of the program and determines how much money should be awarded
    Hint: use some static final constants as return values from your guess-checking method to signal the different kinds of win or loss conditions.
  2. Write appropriate JavaDoc comments for all of your methods and static variables.
  3. Modify your main method create a new kind of game without changing the other methods (perhaps with multiple lottery numbers or more than one guess). Be creative!

Task 2: Class Statistics

Write a program that will process a list of students' grades to calculate a GPA for each student and the class average.

Use methods to organize your program and keep the main method easy to read. For example, you might write separate methods for:

  • Collecting input for (or randomly generating) student names and grades
  • Calculating the average of a single array of numbers
  • Reducing all of the class grades down to a single array of GPA values
  • Printing the list of names and averages

Your program should work with two chunks of data stored as two-dimensional arrays:

  1. Student names, stored as an array of [Lastname, First name] arrays.
  2. Grades for each assignment, stored as an array of arrays of doubles.

Store the data for each student at equivalent indexes in each array. That is, index 0 in one array will store a student's name, and index 0 in the other array will store the grades for that same student.

Start with the following example input to test your program:

Note: Your code should work for any input data that is structured similarly, and should support tables of grades with any number of elements.

If you would like some more grades to test with, try using this code to generate some random data:


			public static double[][] generateMarks(int students, int assignments) {
				double[][] grades = new double[students][assignments];
				for (int i = 0; i < students; i++) {
					for (int j = 0; j < assignments; j++) {
						grades[i][j] = (Math.random() * 10);
					}
				}
				return grades;
			}
