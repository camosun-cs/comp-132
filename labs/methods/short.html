Learning Outcomes and Introduction

In this lab assignment you will learn about:

  • Creating and using methods
  • Passing parameters and receiving return values
  • Splitting code into small reusable components
  • JavaDoc comments

Task 1: Restructure "Days in a Month"

Transform the "Days in a Month" code from Lab 3, Task 1 to use a method.

Your main method should handle all input and output. Move the rest of the code into a new daysInMonth method and use parameters and return values to communicate with the main method.

Make sure to include JavaDoc comments for your new method!

Task 2: Restructure "Play The Lottery"

  1. Recreate your "Play The Lottery" program from Lab 3 Task 2 using methods.

    Your program should be composed of at least 4 different methods:

    • a method that generates a lottery number
    • a method that prompts the user for a guess and ensures that their input is valid
    • a method that compares a guess to a lottery number and determines whether or not the guess was correct (or partially correct)
    • the main method which controls the flow of the program and determines how much money should be awarded
    Hint: use some static final constant numbers as return values from your guess-checking method to signal the different kinds of win or loss conditions.
  2. Write appropriate JavaDoc comments for all of your methods and static variables.
  3. Modify your main method to allow the player to make multiple guesses against the same lottery number. The player should be given a greater reward for matching the lottery number in fewer guesses. Try to limit your changes to the main method; if you can't, look for ways to refactor your other methods so you could use the same code for both versions of the game!

Submission and Grading

Completing all tasks in this lab should result in two separate Java files within a single package. Compress the package directory into a ZIP archive and submit it to the Lab 5 - Methods and Modularization assignment.

Your lab package must be named following the normal convention for labs (LastnameFirstname_lab5), or it will not be marked.
