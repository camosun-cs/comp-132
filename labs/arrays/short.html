Learning Outcomes and Introduction

In this lab assignment you will practice:

  • Creating and modifying arrays
  • Passing arrays as method parameters
  • Traversing multi-dimensional arrays
  • Making aggregate calculations using array values

Task 1: Multiply in Reverse

Write a program that will ask the user to enter a series of integers and then print the result of multiplying each number by all of the numbers that came after it.

For example, given the following sequence of numbers:

7 -5 4 -3 2

Output the result sequence:

840 120 -24 -6 2

Your program should start by asking how many numbers will be entered, then use a loop to collect them from System.in and store them in an array.

Use methods to modularize your code.

Task 2: Class Statistics

Write a program that will process a list of students' grades to calculate a GPA for each student and the class average.

Your program will need two chunks of data stored as two-dimensional arrays:

  1. Student names, stored as an array of [Lastname, First name] arrays.
  2. Grades for each assignment, stored as an array of arrays of doubles.

The two arrays store data for the same student at equivalent indexes. That is, index 0 in both arrays holds data for the first student, index 1 in both arrays holds data for the second, etc.

Use the following example input from your main method to test your program:

Note: Your code should work for any input data that is structured similarly, and should support tables of grades with any number of elements. 

			String[][] names = {
				{ "Ewell", "Nate" },
				{ "Smurthwaite", "Adelina" },
				{ "Bernollet", "Maryjane" },
				{ "Chalfain", "Karlis" },
				{ "Huckabe", "Almire" }
			};
			double[][] grades = {
				{ 3.83, 9.21, 9.25, 0.84, 1.40, 7.90, 1.41, 5.15 },
				{ 5.09, 3.66, 3.04, 3.02, 8.59, 2.34, 8.86, 7.40 },
				{ 5.35, 0.85, 8.56, 7.37, 2.78, 3.19, 2.04, 5.70 },
				{ 9.25, 9.62, 7.08, 7.27, 9.86, 5.88, 9.86, 9.70 },
				{ 3.39, 2.75, 1.23, 1.04, 3.87, 8.72, 4.60, 6.21 }
			};

Using these input values, your program should output:

Student GPAs:
Nate Ewell: 4.87
Adelina Smurthwaite: 5.25
Maryjane Bernollet: 4.48
Karlis Chalfain: 8.57
Almire Huckabe: 3.98

Class average: 5.43
More test data

If you would like some more sample data to test with, use this method to generate additional grades:


			public static double[][] generateMarks(int students, int assignments) {
				double[][] grades = new double[students][assignments];
				for (int i = 0; i < students; i++) {
					for (int j = 0; j < assignments; j++) {
						grades[i][j] = (Math.random() * 10);
					}
				}
				return grades;
			}

You will probably also want to write a method to print this table of grades so that you can check your results!

Improve your code by using methods to organize the separate steps of your program and keep the main method easy to read. For example, you might write separate methods for:

  • Calculating a 1D array of averages from the 2D array of grades
  • Calculating the overall class average
  • Printing the list of names and averages
  • Getting a list of names and grades to process from the user
