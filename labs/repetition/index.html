Task 1: Improved Our Lottery Program

Expand on the lottery program that was designed on the previous lab, make some modifications so that the computer can help the user guess the right number. The computer will generate a random number and give the user an opportunity to guess the right number

  • If the number is too low, the computer will let the user know that the number is too low and prompt the user to guess another number
  • If the number is too high, the computer will let the user know that the number is too high and prompt the user to guess another number
  • When the user guesses the right number, the program ocngradulate the user and let the user know how many trials it took for them to guess the right number, then let them know how much they won, before terminating
    • If the user guess the right number on the first trial, the award is $10,000
    • For any subsequent trial, the award reduces by $1000 per trial
    • Once the award reaches $1000, any subsequent guess will be worth $10 per trial until the user guesses the right number or reaches the maximum.
